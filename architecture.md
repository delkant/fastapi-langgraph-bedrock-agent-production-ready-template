### Library Purpose Summary

#### **Backend Library Purposes**:
1. **FastAPI + Uvicorn**: High-performance async web API framework
2. **LangGraph + LangChain**: AI agent orchestration and LLM abstraction
3. **SQLModel + SQLAlchemy**: Type-safe async ORM with PostgreSQL
4. **Pydantic**: Data validation and serialization
5. **JWT + bcrypt**: Secure authentication and password hashing
6. **Mem0 + PgVector**: Long-term memory with vector search
7. **structlog + Prometheus**: Structured logging and metrics
8. **Tenacity**: Retry logic with exponential backoff
9. **SlowAPI**: Rate limiting for API protection

#### **Frontend Library Purposes**:
1. **Next.js 14 + React 18**: Modern React framework with App Router
2. **TypeScript**: Static typing for better development experience
3. **Tailwind CSS**: Utility-first styling with custom theming
4. **UUID**: Unique identifier generation for messages/sessions
5. **ESLint**: Code quality and consistency enforcement

#### **Key Integration Points**:
- **LangGraph â†” PostgreSQL**: State persistence and checkpointing
- **FastAPI â†” Next.js**: REST API communication with streaming
- **OpenAI/Bedrock â†” LangChain**: Multi-provider LLM integration
- **JWT â†” localStorage**: Session management across frontend/backend
- **Mem0 â†” PgVector**: Long-term memory storage and retrieval
- **structlog â†” Prometheus**: Observability and monitoring stack

This library selection provides a production-ready foundation with:
- **Performance**: Async throughout, connection pooling, optimized builds
- **Type Safety**: TypeScript + Pydantic for end-to-end typing
- **Scalability**: Stateless design with database-backed persistence
- **Observability**: Comprehensive logging, metrics, and tracing
- **Security**: Industry-standard authentication and input validation
- **Resilience**: Retry logic, fallback mechanisms, error handling

- **`_tool_call()`**: Tool execution node that runs available tools
- **`create_graph()`**: Builds the workflow graph with PostgreSQL state checkpointing
- **`get_response()`**: Non-streaming chat completion
- **`get_stream_response()`**: Token-by-token streaming with async generators
- **`get_copilot_stream_events()`**: CopilotKit-compatible event streaming

```typescript
class APIClient {
  private baseURL: string;

  constructor() {
    this.baseURL = process.env.NEXT_PUBLIC_API_BASE_URL || 'http://localhost:8000';
  }

  private async request<T>(endpoint: string, options: RequestInit = {}): Promise<T> {
    const token = AuthStorage.getSessionToken() || AuthStorage.getUserToken();

    const response = await fetch(`${this.baseURL}${endpoint}`, {
      ...options,
      headers: {
        'Content-Type': 'application/json',
        ...(token && { Authorization: `Bearer ${token}` }),
        ...options.headers,
      },
    });

    if (!response.ok) {
      const error = await response.json();
      throw new APIError(error.detail || 'Request failed', response.status);
    }

    return response.json();
  }

  // Streaming implementation
  async *streamCopilotEvents(messages: Message[]): AsyncGenerator<CopilotEvent> {
    const token = AuthStorage.getSessionToken();

    const response = await fetch(`${this.baseURL}/api/v1/chatbot/chat/copilot-stream`, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        Authorization: `Bearer ${token}`,
      },
      body: JSON.stringify({ messages }),
    });

    if (!response.ok) {
      throw new APIError('Streaming failed', response.status);
    }

    // Parse Server-Sent Events
    const reader = response.body!.getReader();
    const decoder = new TextDecoder();

    try {
      let buffer = '';

      while (true) {
        const { done, value } = await reader.read();
        if (done) break;

        buffer += decoder.decode(value, { stream: true });
        const lines = buffer.split('\n');
        buffer = lines.pop() || '';

        for (const line of lines) {
          if (line.startsWith('data: ')) {
            try {
              const data = JSON.parse(line.slice(6));
              yield data as CopilotEvent;
            } catch (error) {
              console.error('Failed to parse event:', line, error);
            }
          }
        }
      }
    } finally {
      reader.releaseLock();
    }
  }
}
```
